## 🔧 Troubleshooting

If you encounter issues, try these common solutions:

*   The application does not open: Ensure you extract the file completely if you downloaded a compressed folder. Try running the application as an administrator by right-clicking the icon.
*   Slow performance: Close background applications that use high amounts of RAM, such as video editors or web browsers with many open tabs.
*   Connection errors: Check your firewall settings. The software needs to send and receive data through standard internet ports.
*   Frozen interface: Click the "Refresh" button in the top menu to reload the dashboard state.
